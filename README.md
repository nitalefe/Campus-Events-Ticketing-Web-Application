# Campus Events Ticketing Web Application

## Project Description
A web-based platform designed to streamline the process of organizing, promoting, and managing campus events. The application enables students and staff to browse upcoming events, purchase tickets, and receive updates, while event organizers can efficiently manage ticketing and attendance.

---

## Objective
The objective of this project is to develop a Campus Events & Ticketing Web Application that centralizes event discovery, management, and participation. The system enhances student engagement through digital ticketing and QR check-in, streamlines event organization for planners, and provides administrators with analytics and oversight tools to improve campus event experiences.

---

## Features

### 1. Student Event Experience
**Event Discovery**
- Browse and search events with filters (date, category, organization).
- View detailed event pages with event information, location, and organizer details.

**Event Management**
- Save events to a personal calendar.
- Claim tickets (free or mock paid).
- Receive a digital ticket with a unique QR code for check-in.
- View all claimed tickets in "My Tickets" section.
- Add events to personal calendar (Google Calendar integration and .ics download).

**Social Features**
- Follow organizers to receive updates on their events.
- Contact event organizers directly.
- View public profiles of organizers.

---

### 2. Organizer Event Management
**Event Creation**
- Create events with comprehensive details: title, description, date/time, location, ticket capacity, and ticket type (free or paid).
- Upload custom event banners and images.
- Edit and update event information.

**Event Analytics**
- Dashboard per event with detailed stats: tickets issued, attendance rates, and remaining capacity.
- Visual charts and graphs for tracking event performance.
- Real-time attendance tracking.

**Attendee Management**
- View complete list of registered attendees.
- Export attendee list in CSV format for external use.
- Track check-in status and attendance.

**Tools**
- Integrated QR scanner for ticket validation at event entry.
- Scan student tickets and verify authenticity.
- Real-time validation feedback. 
---

### 3. Administrator Dashboard & Moderation
**Platform Oversight**
- Approve or reject organizer account applications.
- Review organizer credentials and qualifications.
- Moderate event listings for policy compliance and appropriateness.
- Preview events before they go live.
- Remove or flag events that violate platform policies.

**User Management**
- View all registered users (students, organizers, administrators).
- Search and filter users by role, status, or activity.
- Manage user accounts and permissions.
- Monitor user engagement and activity levels.

**Analytics & Reporting**
- Comprehensive analytics dashboard with global platform statistics.
- View total number of events, tickets issued, and participation trends.
- Track event performance across all organizers.
- Export reports and analytics data.
- Real-time monitoring of platform activity.

**Event Analytics**
- Access detailed analytics for any event on the platform.
- Compare event performance across organizers.
- Identify trending events and popular categories.

**Management**
- Manage organizations and assign roles.
- Configure platform settings and policies.
- Monitor system health and performance.

---

### 4. User Profiles & Social Features

**Public Profiles**
- Public user pages displaying name, photo, bio, and event history.
- View organizer profiles with their hosted events.
- Display follower counts and engagement metrics.

**Following System**
- Follow individual users, organizers, or event categories.
- Receive notifications for followed organizers' new events.
- Unfollow functionality for managing subscriptions.
- View list of followed organizers.

**Personalized Feed**
- Customized feed showing new and upcoming events from followed organizers.
- Filter events based on preferences and following list.
- Discover events tailored to user interests.

**Communication**
- Students receive broadcast messages from organizers and administrators.
- Organizers can broadcast updates to event attendees.
- Organizers receive messages from other organizers and administrators.
- Contact event organizers via email integration.

**Calendar Integration**
- Save events to personal calendar within the platform.
- Export events to Google Calendar.
- Download .ics calendar files for other calendar applications.
